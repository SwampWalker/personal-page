The Chebyshev polynomials

The Chebyshev polynomials form an interesting class of polynomials for use in solving differential equations numerically using pseudospectral methods. The polynomials can be defined by the recurrence relation Although it is tempting to implement such an equation in code using recurrence directly, it is actually better to convert the evaluation to an iteration. The above recurrence relationship can be implemented in Javascript with the following code 

    var ti = x;
    var tiMinus1 = 1;
    for (var i = 1; i < n; i++) {
        var tiPlus1 = 2*x*ti - tiMinus1;
        tiMinus1 = ti;
        ti = tiPlus1;
    }
    return ti;

Interpolation and collocation with Chebyshev polynomials

In pseudospectral methods, functions of interest are approximated as interpolating polynomials. The straightforward definition of a function arises by considering a function to be represented by its coefficients in an expansion over the basis polynomials . This creates an approximation This representation of a function is an approximation due to the truncation of the series at fixed N. Using the tools of analysis, one can represent certain classes of functions using the infinite series representation in terms of the Chebyshev basis . In numerical analysis, the series is generally truncated. Because most functions cannot be represented exactly by a fixed order polynomial, truncating the series leads to errors in the approximation. This source of error may be distinguished from other types of errors and called the truncation error of the approximation. Because most processes that produce an approximate function will not yield the exact values of , there are generally other forms of discretization errors present.

In practice, the so-called coefficient representation of the approximation is sometimes inconvenient to work with. This leads to an alternate representation of the approximation which is referred to as the configuration: . By definition, the configuration is just the value of the approximation evaluated at special points known as collocation points. Two sets of collocation points are frequently used, the roots and the extrema Particular care should be taken in regards to ordering of the points. The extrema end points are often used since this set of collocation points includes the boundaries enabling easy specification of boundary conditions. The roots are useful in cases where regularity is desired: no explicit regularity condition is necessary.

The Chebyshev polynomials

The Chebyshev polynomials form an interesting class of polynomials for use in solving differential equations numerically using pseudo-spectral methods. The polynomials can be defined by the simple recurrence relation Although it is tempting to implement such an equation in code using recurrence directly, it is actually better to convert the evaluation to an iteration. The above recurrence relationship can be implemented in Javascript with the following code 

    var ti = x;
    var tiMinus1 = 1;
    for (var i = 1; i < n; i++) {
        var tiPlus1 = 2*x*ti - tiMinus1;
        tiMinus1 = ti;
        ti = tiPlus1;
    }
    return ti;
